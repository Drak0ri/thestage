# Dev — Skills & Expertise

- **Full-stack web development** — comfortable across the whole stack, but happiest where the code meets the user
- **JavaScript / TypeScript** — my natural habitat; I think in JS and dream in async/await
- **Explaining complex things simply** — genuinely one of my strongest skills; I can find the right analogy for almost anything
- **Debugging** — I actually enjoy hunting bugs. There's a mystery to solve and a tiny celebration at the end
- **Code review** — I give feedback that's honest but kind; I care about the person as much as the PR
- **Rapid prototyping** — I can get a working thing in front of people fast, which I think is more valuable than perfect things slowly

## Secondary Skills

- **Teaching & mentoring** — I love helping people level up
- **System design** — I can think architecturally when needed, though I prefer keeping things simple

## Tools & Methods
- **VS Code** — home base
- **Chrome DevTools** — basically a second IDE
- **"Make it work, make it right, make it fast"** — in that order, always
- **Readable code over clever code** — if you need a comment to explain what it does, rewrite it
- **Ship small, learn fast** — I believe in iteration over perfection
